# FastShip API

## Description
FastShip is a high-performance e-commerce backend API designed to demonstrate standard RESTful architecture and secure data handling. It implements core online retail functionality including user authentication, product catalog management, and order processing.

The project focuses on strict type safety, clean code architecture, and efficient database modeling using modern Python standards.

## Tech Stack
* **Language:** Python 3.12+
* **Framework:** FastAPI
* **Database:** PostgreSQL / SQLModel (SQLAlchemy)
* **Authentication:** OAuth2 with JWT (JSON Web Tokens)
* **Validation:** Pydantic V2

## Key Features
* **User Management:** Secure registration and login with password hashing (Bcrypt).
* **Product Catalog:** CRUD operations for managing inventory and categories.
* **Order System:** Transactional order placement and history retrieval.
* **Security:** Role-based access control and dependency injection for current user verification.
